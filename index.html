Langkah-langkah pembuatan github page menggunakan sistem git,

  • Login kedalam akun github yang telah dibuat/ yang telah dimiliki.
  • Buat repositori baru dengan nama (username kita).github.io
  • Lakukan git bash here dengan klik kanan di dalam folder file html yang telah dibuat.
  • Cek status dengan "git status" lalu lakukan remote terhadap repository di github.
  • Lakukan commit file html dan kawan-kawannya dengan "git commit"
  • Setelah itu lakukan push file tersebut dengan "git push" ke dalam repositori github.
  • Pergi ke bagian settings di repository github yang telah dibuat, lalu pilih pages dan ganti branch dari none menjadi main
  • Tunggu beberapa saat dan lihat hasil akhirnya
